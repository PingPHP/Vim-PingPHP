# Vim-PingPHP
Vim Plugin Of PingPHP 

## Feature
- syntax highlighting
- indent
- omnicomplete function

## Install

Use [Vundle][] or [pathogen.vim][] is recommended.

[Vundle]:https://github.com/gmarik/vundle
[pathogen.vim]:https://github.com/tpope/vim-pathogen

For Vundle:

    Plugin 'PingPHP/Vim-PingPHP'
